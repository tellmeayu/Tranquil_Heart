# Tranquil Heart
An interactive mindfulness app that enables users to create calming soundscapes through touch and gesture controls. Built with Swift for iOS/iPadOS.

## Overview
Tranquil Heart provides an active approach to meditation through three interactive mini-games where users manipulate audio and haptic feedback in real-time. Rather than passive listening, users engage directly with sound to achieve focus and relaxation.

## Features
- **Touch for Music**: Tap-based harmonic tone generation
- **Harmony with Me**: Swipe-controlled tremolo effects
- **Feel Your Vibe**: Haptic feedback synchronized with gestures
- Minimalist UI design
- Real-time audio synthesis and manipulation

For more details: https://www.sylviastudio.cn/tranquil-heart-overview/

A demo video of one of the mini games (Harmony With Me): https://www.bilibili.com/video/BV1ow7tziEZn


## Requirements
- iOS/iPadOS 15.0+
- Swift 5.5+
- Swift Playgrounds 4.0+ or Xcode 16.0+

## Getting Started
1. Open `Tranquil Heart.swiftpm` in Swift Playgrounds or Xcode
2. Build and run the project
3. Select a mini-game from the home screen
